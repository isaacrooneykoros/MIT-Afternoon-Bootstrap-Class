What's the Difference?

HTML

Hypertext Markup Language
Create the structure
  • Controls the layout of the content
  • Provides structure for web page design
  • The fundamental building block of any web page

CSS

Cascading Style Sheets
Style the website
  • Applies styke to elements
  • Targets various screen sizes for responsiveness
  • Handles the look and feel of a page

JavaScript

Increase interactivity
  • Adds interactivity to pages
  • Handles complex functions
  • Makes web pages dynamic and interactive
